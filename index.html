API Basejump: Image Search Abstraction Layer

用户故事： 根据指定的字符串描述，获取一组符合描述的图片地址和文字描述以及页面地址。
用户故事： 可以在地址中增加一个参数 ?offset=2 来对返回结果进行分页。
用户故事： 可以获取一个列表，显示最近几次搜索的搜索语句和返回页面地址。

How to use

'/latest/imagesearch/' can lead you to history keywords.

'/imagesearch/keyword?offset=4' can lead you to search result.
